Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, we used flip-flops and multiplexers to create 
registries that can store keypad inputs. Data can be read and written 
to and from these registries.
-----------
FILES

-
Lab2.lgi

This file contains the assembly code of the lab.

----------
INSTRUCTIONS

-
Use the keypad to generate numbers, and flip the switches of the read 
and write operators to store data in or obtain data from the registries.
